# mPCRselect  
Pipeline for selecting ancestry and individual identification informative SNPs from variant call files (VCF).  

## Dependencies  
mPCRselect requires the following software:  

* Nextflow v. >= 23.10.0 [1]  
* Ruby v. >= 3.2.2 [2]  
* VCFtools v. 0.1.16 [3]  
* gzip  
* Python v. >= 3.0  
* BEDtools v. >= 2.31.0 [4]
